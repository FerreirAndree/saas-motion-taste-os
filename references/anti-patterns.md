# Anti-Patterns

## Generic Tech Aesthetic

Symptoms:

- floating rounded rectangles with no product logic
- purple-blue gradient background by default
- random glowing lines
- meaningless particles
- fake charts with no narrative
- "AI" represented only as sparkles or orbs

Fix:

- Rebuild visuals from the product's actual workflow, data, decisions, and user outcomes.

## Constant Motion

Symptoms:

- every object drifts
- camera never rests
- UI moves while text must be read
- no meaningful holds

Fix:

- Assign each motion a job. Freeze or remove secondary movement during proof beats.

## Equal-Weight Everything

Symptoms:

- all cards are same size
- all labels are same contrast
- all motion has same speed
- every feature receives equal emphasis

Fix:

- Choose a hero idea for each shot. Reduce everything else to context.

## Fake Product Syndrome

Symptoms:

- UI is too abstract to prove anything
- labels are vague
- charts are decorative
- workflows have no real inputs or outputs

Fix:

- Use plausible product states, realistic labels, and visible before-after changes.

## Template Transitions

Symptoms:

- repeated slide-ins
- unrelated fades
- whip pans with no spatial logic
- scenes connected only by timing

Fix:

- Use match cuts, object expansion, state transformation, or attention handoff.

## Overexplained Copy

Symptoms:

- full sentences everywhere
- voiceover and text say the same thing
- type competes with UI
- viewer cannot read before the cut

Fix:

- Reduce copy to claims, labels, and proof points. Let product motion carry explanation.

## Premature Logo

Symptoms:

- logo appears before the viewer understands the problem
- brand animation interrupts narrative
- product entrance has no dramatic reason

Fix:

- Build tension or desire first. Introduce the product as the organizing answer.

## Decorative 3D

Symptoms:

- camera orbit exists only to look premium
- product surfaces are unreadable
- depth makes the idea less clear

Fix:

- Use depth to reveal hierarchy, scale, or system relationships.

## Feature Soup

Symptoms:

- rapid montage of unrelated capabilities
- no sequencing logic
- each shot starts from zero

Fix:

- Group features into narrative chapters or show one workflow where capabilities compound.

## No Trust Beat

Symptoms:

- video shows speed and magic but no control
- enterprise claims feel risky
- AI output appears without verification

Fix:

- Add permissions, auditability, review, sources, reliability, deployment, or human control.
